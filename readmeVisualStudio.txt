README for Visual C++ 6.0 and later

In Visual C++ or Visual Studio .NET, open the file
SampleCode\VisualStudio60\OGLDSamples.dsw. If using a more recent
version than 6.0, convert all project and workspace files when
prompted to do so. After the projects load, under the Build manu,
select Batch Build, and then Rebuild All.
